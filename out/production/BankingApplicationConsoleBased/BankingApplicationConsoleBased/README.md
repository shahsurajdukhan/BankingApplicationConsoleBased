# 🏦 Banking Application (Console-Based)

A simple **console-based banking application** built using Java. This project demonstrates core banking functionalities and object-oriented programming concepts.

---

## 📌 Features

* 🧾 Create new bank account
* 💰 Deposit money
* 💸 Withdraw money
* 📊 Check account balance
* 🔐 Basic user interaction via console

---

## 🛠️ Tech Stack

* Java
* OOP (Object-Oriented Programming)
* Console-based interface

---

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Tanuj325/BankingApplicationConsoleBased.git
```

### 2️⃣ Open in IntelliJ IDEA

* Open IntelliJ IDEA
* Click on **Open Project**
* Select the project folder

### 3️⃣ Run the project

* Navigate to main class (e.g., `Main.java`)
* Click **Run ▶️**
